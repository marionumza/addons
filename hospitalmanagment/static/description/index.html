Manage Patient details

Patient can enter their details.There are tabs will shows patient details like medicament, diseases, lab test, etc.And can print patient card,patient medical history report,patient diseases report and vaccination history report.

Receptionist can Create Appointment

Receptionist can create appointment of patient with the details like physician , consultation service with date and time etc and can see appointment list ,cancel appointment and done appointment

Generate Patient's Prescriptions

Generate prescriptions with the doctor's and patient's details, can create invoice from prescription.

Lab Test Request and Result

Patient can request fot the lab test and can get the result of the lab test.Medical group and chemist can access it

User can Generate Report

User can generate the report for their details, appointment, prescriptions, lab-test,imaging test etc. And also patient medical history and vaccination detail etc
