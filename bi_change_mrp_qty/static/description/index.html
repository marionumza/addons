Change Finished Product Quantity in Work Order-Food Manufacturing Process Odoo App

Manufacturing flexible finished goods quantity odoo app helps user to change finished product quantity in manufacturing's work order process. Production user can increase or decrease finished product quantity from workorder screen and produce more or less product quantity for production order via work order and it consumed raw material same as planned Quantity . This odoo apps mostly use for food manufacturing industry where user don't know what will be actual finished good quantity after finished the manufacturing order. By default Odoo doesn't allow to record production for more or less than plan produce quantity with same consumed materials, Our this odoo module solved that problem so production user can record produce quantity as per actual output they receive after that production which consumed material as per plan quantity . If you have multiple work order on manufacturing order then change qty option will be appears on the last work order screen so once you finished the record production it will logged added finished goods quantity and finished manufacturing order.
